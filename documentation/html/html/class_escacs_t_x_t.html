Efectuar una tirada.

Precondició
p != null i input té el format d'una jugada ordinària o enroc.
Postcondició
S'efectua la tirada input a la partida p. Si input té format d'una jugada ordinària, es realitza una jugada ordinària, altrament, una jugada enroc. També es demana i s'efectua la promoció en cas d'haver-n'hi. Es retorna un booleà que és cert si cal acabar la partida.
Paràmetres
pPartida en què s'efectuarà la tirada i la promoció (en cas d'haver-n'hi).
inputTirada que es vol efectuar.
Retorna
Un booleà que és cert si la partida ha finalitzat i cal gestionar l'acabament de la partida, fals altrament.
Excepcions
ExcepcioJugadaErroniaSi la situació de la partida i/o el tauler no permeten efectuar la jugada.

◆ demanarCPU()

static Pair<Boolean, Boolean> EscacsTXT.demanarCPU ( )
staticprivate

Demanar quins jugadors són humans i quins són controlats per la CPU.

Precondició
—
Postcondició
Es demana per pantalla que s'esculli si el jugador de blanques i/o de negres han d'estar controlats per la CPU. En cas d'entrar una s, el jugador indicat passarà a realitzar tirades i respondre a les taules de forma automàtica. En cas d'entrar n, el jugador s'interpreta com a humà i se li demanaran les tirades. Es retorna un Pair amb un booleà que indica si el jugador de blanques és CPU i un altre booleà que indica si el jugador de negres és CPU.
Retorna
Un Pair amb dos booleans, el primer indica si el jugador de blanques és CPU, el segon si ho és el de negres.
