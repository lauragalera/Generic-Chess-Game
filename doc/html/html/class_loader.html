Aplicar les tirades a la partida.

Carregar una partida a partir d'un fitxer de desenvolupament de partida.

Precondició
path és la ruta a un fitxer JSON vàlid i conté el desenvolupament d'una partida amb un format vàlid.
Postcondició
Es retorna una Partida carregada amb el fitxer de desenvolupament path. La partida estarà just en el punt en què es va ajornar o començarà amb les posicions inicials i el torn indicats.
Paràmetres
pathRuta a un fitxer de dades JSON.
Retorna
Una partida carregada amb el fitxer de desenvolupament.
Excepcions
IOExceptionSi el fitxer no és un JSON o no es pot obrir.
IllegalStateExceptionSi la partida ja està finalitzada.
IllegalArgumentExceptionSi el format del fitxer no és correcte, les dades contingudes no són coherents o els tipus de dades no són correctes.

Iniciar una partida a partir d'un fitxer de regles.

Precondició
path és la ruta a un fitxer JSON vàlid i conté les regles d'una partida amb un format vàlid i 0 <= properTorn <= 1
Postcondició

Paràmetres
pathRuta a un fitxer de regles JSON.
partidaNovaIndica si la partida és nova o es reprèn el joc.
properTornIndica a quin jugador li toca tirar a continuació.
Retorna
Una partida iniciada amb el fitxer de regles.
Excepcions
IOExceptionSi el fitxer no és un JSON o no es pot obrir.

Llegir les peces.

Precondició
peces != null i enrocsDisponibles != null. A peces, no hi pot haver peces amb el mateix nom i el valor de totes elles ha de ser un enter positiu. El rei ha de ser únic, vulnerable i ha de tenir el valor més alt. Les dades tenen el format correcte.
Postcondició
Es llegeixen i comproven les peces. Es crea un mapa de nomPeca,Peca i es retorna. Les peces contingudes al mapa tenen s'han carregat amb totes les dades llegides, inclosos els enrocs definits a enrocsDisponibles.
Paràmetres
pecesLlista que conté els mapes que defineixen les peces.
enrocsDisponiblesEnrocs disponibles per cada peça juntament amb quina peça el realitzen i l'enroc en sí.
Retorna
Un mapa amb les peces creades.
Excepcions
